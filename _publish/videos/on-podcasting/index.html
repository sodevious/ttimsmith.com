On Podcasting

22 Feb 2012

Wanted to create a quick screencast that goes over the basics on setting up Podpress, a plugin for Wordpress that is made for podcasting. We go through creating a feed, running it through feedburner and submitting to the iTunes store. Hope you find this useful!
